# LucidNN

**LucidNN** is an interactive neural network visualization tool built with **Streamlit**. It enables users to design, inspect, modify, and simulate feed-forward neural networks through an intuitive graphical interface.

The application is designed as an educational tool for understanding neural network architecture, weights, biases, and training behavior.


## Features

* Build custom neural network architectures
* Interactive visualization of network topology
* Edit neuron weights and biases manually
* View layer-wise network statistics
* Simulate neural network training
* Monitor error reduction across epochs
* Perform weight pruning using configurable thresholds
* Import pretrained models

  * PyTorch (`.pth`, `.pt`)
  * Keras (`.h5`, `.keras`)
  * LucidNN JSON (`.json`)
* Export models in LucidNN JSON format


## Tech Stack

* Python
* Streamlit
* NumPy
* Pandas
* Graphviz
* PyTorch
